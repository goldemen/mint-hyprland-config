# Hyprland Workstation Configuration

Configuration minimale et reproductible d un environnement Linux base sur :

- Linux Mint
- Hyprland Wayland compositor
- Waybar
- SwayNC
- Wlogout
- Theming GTK et Qt

Ce depot contient uniquement la configuration niveau systeme et bureau.

# Environnement

- **Systeme :** Linux Mint
- **Serveur graphique :** Wayland
- **Compositeur :** Hyprland
- **Barre de statut :** Waybar
- **Notifications :** SwayNC
- **Menu logout :** Wlogout
- **Theme :**
    - GTK 3
    - Qt5ct
    - Qt6ct
    - Kvantum
    - Wallust

## Structure du depot

Tous les dossiers correspondent au contenu habituellement situe dans **~/.config/**
